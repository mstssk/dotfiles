# dotfiles

今の所、mac前提

## 使い方

※ setup.sh が Homebrew https://brew.sh/ をインストールして、各種アプリをインストールします。

```
$ cd
$ git clone git@github.com:mstssk/dotfiles.git $HOME/dotfiles
$ cd $HOME/dotfiles
$ ./setup.sh
$ zsh
```

zshをターミナルのデフォルトに設定するなど別途必要
